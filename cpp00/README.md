# cpp00

## Overview

This project was my first contact with C++ in the 42 curriculum. It introduced me to the fundamentals of the language and helped me transition from procedural programming (C) to a more structured and modular approach.

## What I Learned

### Object-Oriented Programming

One of the most important concepts I learned was **Object-Oriented Programming (OOP)**.
Instead of writing everything as functions, I started organizing code into **classes and objects**, which made programs:

* more modular
* easier to read
* easier to maintain

I learned how to:

* define a class
* create objects
* use methods (member functions)

### Using Existing C++ Functions

Another key lesson was understanding how powerful C++ is thanks to its **standard library**.

Instead of reinventing everything, I learned how to:

* use built-in functions
* explore existing libraries
* combine them creatively to solve problems

### Public vs Private Attributes

I also learned about **encapsulation**, using:

* `public`
* `private`

This allowed me to:

* protect internal data of a class
* control how attributes are accessed and modified
* write safer and more robust code

It introduced me to the idea that not everything in a class should be directly accessible.

## Final Thoughts

CPP00 establishes the foundation of C++ by introducing OOP structure, basic class design, and effective use of the standard library.
