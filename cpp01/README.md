# cpp01

## Overview

This project deepened my understanding of C++ by focusing on memory management and the different ways data can be accessed and shared. It helped me move beyond basic class usage and start thinking more carefully about how data is stored and manipulated.

## What I Learned

### Memory Allocation

One of the main concepts introduced in this module was dynamic memory allocation.

I learned the difference between:

```
stack allocation
heap allocation
```

And how to use:

```
new
delete
```

This helped me understand how memory works behind the scenes and how to avoid issues like memory leaks.

### References and Pointers

This module also clarified the difference between references and pointers.

I learned:

```
how references act as aliases to existing variables
how pointers store memory addresses
when to use each one
```

Understanding this distinction made it easier to write cleaner and more efficient code.

### Switch Statements

Another concept introduced was the use of switch statements as an alternative to multiple if/else conditions.

This helped me:

```
simplify control flow
make code more readable in certain situations
```

### File Streams

I was introduced to file handling in C++ using streams.

I learned how to:

```
read from files
write to files
manipulate file content
```

This opened the door to working with external data instead of just console input/output.

## Final Thoughts

CPP01 builds on the basics of C++ by introducing memory management, references, and file handling. It strengthened my understanding of how data is handled in C++ and prepared me for more advanced concepts in later modules.
