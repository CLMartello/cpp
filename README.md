# C++

This repository contains my solutions for the **C++ Modules** from the 42 curriculum.

The goal of these modules is to build a solid understanding of **Object-Oriented Programming (OOP)** in C++, while progressively introducing more advanced concepts such as memory management, operator overloading, inheritance, polymorphism, and templates.

---

## Modules Overview

| Module    | Topics Covered                                             | 
| --------- | ---------------------------------------------------------- |
| **CPP00** | Basics of C++, classes, member functions, std::string      |
| **CPP01** | Memory allocation, references, pointers, switch statements |
| **CPP02** | Ad-hoc polymorphism, operator overloading, canonical form  |
| **CPP03** | Inheritance                                                |
| **CPP04** | Subtype polymorphism, abstract classes, interfaces         |
| **CPP05** | Exceptions                                                 |
| **CPP06** | C++ casts                                                  |
| **CPP07** | Templates                                                  |
| **CPP08** | Templated containers, iterators                            |
| **CPP09** | STL (Standard Template Library)                            |

---

## 🧩 Key Concepts Learned

* Class design and encapsulation
* Orthodox Canonical Form (Rule of Three)
* Operator overloading
* Inheritance and polymorphism
* Exception handling
* Type casting
* Templates and generic programming
* STL containers and algorithms

---

## Compilation

All modules are compiled using:

```bash
c++ -Wall -Wextra -Werror -std=c++98
```

---

## How to Run

Navigate to a specific module and exercise:

```bash
cd cpp02/ex02
make
./program_name
```
